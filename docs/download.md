# 下载 WorldCupIssue

### v0.1.11 最新版本

当前复刻版安装后生成 `world_cup_issue.exe`，应用版本为 `0.1.11`，用户数据目录为 `com.worldcupissue.app`。

## 系统要求

| 平台 | 最低版本 | 说明 |
| --- | --- | --- |
| Windows | Windows 10 / 11 x64 | 第一版优先保证 |
| macOS | 预留 | 后续验证 |
| Linux | 预留 | 后续验证 |

## 安装说明

Windows 使用 NSIS 安装包。关闭窗口默认收进系统托盘，需要从托盘菜单退出。

## 自动更新

复刻版保留检查更新入口。发布新的签名安装包时，将 `latest.json` 与 updater zip 上传到 GitHub Release 后，桌面端可以通过“检查更新”发现并自动升级。
