## 一句话理解

WorldCupIssue（世界杯组手）是复刻杯况 CupWatch 体验的桌面盯分工具：你忙你的，它替你盯着世界杯比分。

## 核心特色

- 中文 + 北京时间
- 托盘常驻 + 系统通知 + 悬浮比分条
- 公开数据源优先，本地缓存兜底
- 纯资讯，无投注，与 FIFA 及官方转播机构无关
